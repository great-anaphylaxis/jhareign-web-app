Operating System by Jhareign Solidum (2022)

This is my attempt at creating my own operating system. It is a 32 bit operating system with it's own bootloader. Note: It has not been tested on real hardware so it MIGHT NOT WORK ON REAL HARDWARE. But, it works on QEMU (a virtual machine).

The project can be downloaded here

I use this to run the bin file

"C:\Program Files\qemu\qemu-system-x86_64.exe" -soundhw pcspk,sb16 "os.bin"
